swiperRequireFocus
directive in module ionic.swiper

Description

This directive makes a 'slide to me' request after all the other slides have been added You will probably not use it, because the user can see the sliding transition If you dont want this to happen, you can use the swiper-slide with instructions like swiper-slide="left|right|center"

Dependencies

  • $timeout

Usage

as attribute
<ANY swiper-require-focus>
   ...
</ANY>
