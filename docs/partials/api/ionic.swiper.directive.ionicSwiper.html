ionicSwiper
directive in module ionic.swiper

Description

An alternative for swiperContainer using transclude but limited to 3 slides

It has some properties that you can use:

left-swiper and right-swiper are used to validate if, even with transclude slots fullfiled, they should be used as swipable containers. Those options do not create watchers and eval only once, if they are not provided the default value is true.

is-swipable creates an watcher and updates swiper container with no-swipe-class of swiper (or the custom set in swiper configs provider)

center-on-disable if the directive should slide to central slide when swiper is disabled. This one defaults to true and does not create watcher.

Dependencies

  • $rootScope
  • $parse
  • SwiperService
  • SwiperConfigurationsProvider

Usage

as element:
<ionic-swiper>
</ionic-swiper>
